The text align is a type of propertie which is used weather the text should be in center or right or left.
        The example of the text align is 
        how can we defind by using the element selector is 
        h1{
            textalign: center or left or right;
        }
        or else
        by using the class selector is(the selector is depend on the what name you are giving to the attribute) 
        .center{
            text align:center or right or left;

        } 
        by using the id selector is(the selector is depend on the what name you are giving to the attribute)
        #center{
            text align:center or right or left;
        }
